# Computer_Vision_ESIGELEC_AES_2026
Class work for the course of Computer Vision at ESIGELEC in 2026



The code presented here is done with the aid of chatgpt
works in windows only

*test_1 is displaying an image using opencv
*test_2 is using the videostream to display the video using opencv
*test_3 is detecting red pixels from the videostream and drawing a green dot on the centroid of      the red pixels
*test_4 is a game using a green object as a brush in realtime to draw on screen and pop bubbles.
*test_5 is an detection alarm program 
*test_6 is an invisibility cloak program that turns clumps of black pixels invisible by maskin the   background pixels on it.
